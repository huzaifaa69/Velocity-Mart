 **Velocity-Mart**

Velocity-Mart is a Warehouse Slotting Optimization Web Application built using Python and Streamlit.
It analyzes warehouse order data, SKU master data, and picker movement to generate an optimized slotting plan for improved operational efficiency.

**Project Overview**

Velocity-Mart helps in:

Analyzing order transaction data

Evaluating SKU movement and demand

Studying picker travel patterns

Applying warehouse constraints

Generating an optimized slotting plan

The goal is to improve warehouse productivity by reducing picker travel time and placing high-velocity SKUs in optimal locations.
